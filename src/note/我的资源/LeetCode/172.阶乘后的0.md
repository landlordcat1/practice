给定一个整数 n，返回 n! 结果尾数中零的数量。

```
示例 1:

输入: 3 输出: 0 解释: 3! = 6, 尾数中没有零。
```


```

示例 2:

输入: 5
输出: 1
解释: 5! = 120, 尾数中有 1 个零.
```
说明: 你算法的时间复杂度应为 O(log n) 。
解决方法：
```
找2,5,10的倍数， 因为5的倍数最少，所以只用找5的倍数 特殊情况25 有两个5,125有3个5，以此类推
```
只要有5，就会有2，就会有10。
所以末尾就会有0。
```
class Solution {
    public int trailingZeroes(int n) {
    int num = 0;
    long temp = 5;
    while(n > 0){
        num += n / 5;
        n /= 5;
    }
        return num;
    }
}
```
